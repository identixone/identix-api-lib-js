## Installation

### NPM

```bash
npm i --save identix-api-lib-js
```

### Yarn

```bash
yarn add identix-api-lib-js
```

## Usage

```js
// ES2015 module import:
import IDXApi from "identix-api-lib-js";

// CommonJS module require:
const IDXApi = require("identix-api-lib-js");
```
